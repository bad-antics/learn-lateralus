# learn-lateralus

Structured textbook-style chapters for learning Lateralus from scratch.

## Chapters

| Chapter | Title |
|---------|-------|
| 1       | Hello, World! and Basics |
| 2       | Variables and Types |
| 3       | Functions and Closures |
| 4       | Pattern Matching |
| 5       | Structs and Enums |
| 6       | Collections |
| 7       | Error Handling |
| 8       | Generics and Traits |
| 9       | Lifetimes and Borrowing |
| 10      | Modules and Packages |
| 11–15   | Async, Iterators, Concurrency, Testing, Documentation |
| 16–20   | Closures (advanced), Custom Iterators, Concurrency, FFI, Performance |
| **21**  | **Macros and Metaprogramming** |
| **22**  | **Unsafe Lateralus** |
| **23**  | **The Type System in Depth** |
| **24**  | **Embedded / no-std** |
| **25**  | **Building a CLI Tool End-to-End** |

## File count: **40 `.ltl` files**

## Running

```bash
ltl run chapters/01-hello/chapter01.ltl
ltl run chapters/21-macros/chapter21.ltl
```

Each chapter has a main file and an `exercises` file with hands-on problems.
